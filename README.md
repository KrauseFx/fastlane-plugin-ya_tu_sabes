## Getting Started

This project is a [fastlane](https://github.com/fastlane/fastlane) plugin. To get started with fastlane-plugin-ya_tu_sabes, add it to your project by running:

```bash
fastlane add_plugin ya_tu_sabes
```

## About ya_tu_sabes
